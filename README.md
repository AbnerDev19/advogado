
📌 Vitor Rocha Advocacia — Site Institucional + Painel Administrativo

📖 Sobre o projeto

-Este projeto é um site institucional desenvolvido para o escritório Vitor Rocha Advocacia, com foco em transmitir credibilidade, profissionalismo e autoridade jurídica desde o primeiro acesso.

-Além da página pública, o sistema conta com um painel administrativo interno, permitindo a gestão de contatos (leads) e publicação de conteúdos jurídicos.

🚀 Funcionalidades

🌐 Site Institucional
Página inicial moderna e responsiva
Seções:
-Apresentação do escritório
-Áreas de atuação
-Diferenciais
-Conteúdo jurídico
-Formulário de contato
-Alternância de tema (claro/escuro)
-Layout focado em experiência do usuário e conversão

📰 Página de Conteúdo
-Exibição de artigos e notícias jurídicas
-Estrutura preparada para fortalecer autoridade digital
-Destaque para conteúdos principais

🔐 Sistema de Login
-Área restrita para acesso administrativo
-Validação de credenciais (front-end)

📊 Painel Administrativo
-Gestão de contatos recebidos pelo site:
-Visualização de leads
-Filtros por status (novo, em andamento, concluído)
-Contadores dinâmicos
-Gestão de conteúdo:
-Criação de novos artigos
-Listagem de conteúdos publicados

🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla)
Google Fonts (Inter + Playfair Display)

⚙️ Como executar o projeto
Baixe ou clone este repositório
Abra o arquivo index.html no navegador

Não é necessário servidor backend — projeto 100% front-end

📌 Observações importantes
-O formulário de contato e login são simulados (não possuem integração com backend)
-Os dados do painel (leads e conteúdos) são manipulados via JavaScript
-Estrutura pronta para futura integração com:
-API
-Banco de dados
-Sistema real de autenticação

🎯 Objetivo do projeto

-Criar uma presença digital profissional, moderna e estratégica, alinhada com a imagem de um escritório jurídico de alto padrão, além de oferecer uma base sólida para evolução futura do sistema.

📈 Possíveis melhorias futuras
-Integração com backend (Node.js, PHP ou Firebase)
-Autenticação real de usuários
-Banco de dados para leads e conteúdos
-Upload de imagens para artigos
-SEO avançado
-Integração com WhatsApp e CRM
